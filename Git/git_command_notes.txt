git 命令相关笔记

0. 概念
在Git中，用HEAD表示当前版本
上一个版本就是HEAD^，上上一个版本就是HEAD^^
当然往上100个版本，写成HEAD~100

1.常用命令：
git init

git add
git commit -m 'message'
git diff 
git push
git log 
    查看历史纪录
git log --pretty=oneline 
git reflog 查看log 包括退回版本ID
